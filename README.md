## Air Ticket Reservation System

A full-stack **Air Ticket Reservation System** built using **React** for the frontend and **Spring Boot microservices** for the backend. The application enables secure booking and management of airline tickets with scalable and robust service communication.

### Features
- React-based frontend for a responsive and user-friendly UI  
- Spring Boot microservices architecture for modular and scalable backend services  
- JWT-based authentication for secure user login and session management  
- Role-based authorization to control access for different user roles (Admin, User, etc.)  
- API Gateway for centralized routing, security, and communication between services  
- Global exception handling for consistent and meaningful error responses  
- JUnit testing for ensuring code quality and reliability  

### Tech Stack
- **Frontend:** React, JavaScript, HTML, CSS  
- **Backend:** Spring Boot, Spring Security, Spring Cloud  
- **Authentication:** JWT (JSON Web Tokens)  
- **Testing:** JUnit  
- **Architecture:** Microservices with API Gateway
